# Recruiter-Facing Explanation

Atlas is a full-stack local deterministic prototype of a personal AI operating system for
engineering, learning, and career growth, with LLM-ready architecture. It combines a polished Next.js
workspace with a FastAPI backend, persistent memory, retrieval with citations, trace logging,
workflow execution, approval-gated tools, repository ingestion, static code intelligence, graph
visualization, risk analysis, provider abstractions, and a local evaluation suite.

## What Makes It Different

- It is context-aware: Atlas uses profile, resume, memory, journal, project, and codebase evidence.
- It is traceable: chat, workflows, evals, and approved tools create durable trace records.
- It is safer than autonomous agents: write actions are previewed and require approval.
- It is product-shaped: the UI has dedicated surfaces for workflows, memory, code intelligence,
  actions, traces, and evaluations.
- It is extensible: the current local engine has provider boundaries for OpenAI-compatible,
  Ollama, and vLLM-style models, and can later add LangGraph or Agents SDK orchestration,
  tree-sitter grammars, and external connectors.
