# Screenshot Checklist

Save polished demo screenshots in this folder after starting the local app.

Recommended captures:

- `01-command-center.png`: dashboard priorities, metrics, current projects, approvals, recent traces,
  weak areas, and voice command panel.
- `02-code-intelligence.png`: repo selector, React Flow graph, searchable symbols, and risk report.
- `03-actions.png`: proposed action preview with approve/reject buttons and artifact audit.
- `04-traces.png`: trace detail with retrieved memory, prompt version, tool calls, assumptions, and
  steps.
- `05-evals.png`: evaluation prompts and latest run results.

Use the demo flow in `../demo-video-script.md` as the capture order.
